写入U盘

  1. boot 引导分区: 300~500M /boot
  2. / 根目录分区: 大于20g
  3. /home 用户目录分区: 剩余空间
  4. swap 交换分区: 可以不用分，如果内存低于4g设置为内存的2倍大小

注意 boot和/分区是必须要有的，其它分区可无。
