win10子系统linux安装

发表于 2017-09-26 | 更新于 2021-08-11 | 分类于 Linux | 阅读次数：
本文字数： 12k | 阅读时长 ≈ 11 分钟

介绍

微软在Windows10 1607版中提供了一个Linux子系统，用户不需要虚拟机就可以直接运行Linux命令，是一个对开发者非常方便的功能。目前服务器领域基本都是Linux的天下，但是在个人桌面领域，windows还是独大。可怜的开发者或者使用虚拟机或者用windows上阉割版的程序。有了Linux子系统，苦逼的开发者可以不离开舒服的windows环境，又可以直接使用Linux强大的各种开源软件，听起来是一个两全的方案。
Windows subsystem for linux（wsl）是基于Ubuntu的，类似于cygwin的实现方案，看起来可以支持除了desktop外的所有功能。Linux的desktop本来就不强，强大的是那些开源工具和产品。
实际试用下来，速度有点点慢，尝试装了下java，慢的无法忍受。但整体还是很满意。

windows 10 1703（创新者更新）提供了新的Ubuntu 16.04， 建议大家用 lxrun重新安装wsl

安装

启用linux子系统

在程序与功能中，选择启用或者关闭Windows功能，然后勾上“适用于LInux的Windows子系统”后确定即安装。如下图所示：

允许开发者功能

在更新和安全中，打开开发人员模式。如下图
注意:有可能会看到一个 开发人员模式程序包无法安装.错误代码0x80004005 ,出现这个问题一般不影响后续步骤,但是建议运行Windows 更新,获取最新的更新包

注意,需要打开windows update服务
您可以尝试以下操作，删除Windows 更新缓存文件：
按Windows键+X，选择“命令提示符（管理员）”；
输入：net stop wuauserv，回车（此处会提醒服务停止）；
在C:\Windows\SoftwareDistribution文件夹中，找到并删除Download和DataStore文件夹中的所有文件；
输入：net start wuauserv，回车（此处会提醒服务开启）；
最后在Windows Update中，尝试检查更新。

安装细节

  1. 系统升级到一周年正式版及以上(1607)
  2. 依次在设置-更新与安全-针对开发人员选项中，启用”开发人员模式”
  3. 在资源管理器中打开控制面板\所有控制面板项\程序和功能,打开”启用或关闭 Windows功能”,勾选”适用于Linux的Windows子系统(Beta)”
  4. 重启电脑
  5. 命令行运行lxrun /install /y(或输入bash)开始安装
  6. 安装速度取决于网络情况，下载的文件在%localappdata%\lxss目录下lxss.tar.gz(181M)，解压后大概500M,rootfs目录即为子系统根目录。
  7. 命令行运行bash进入Ubuntu
  8. 默认使用的root帐号登录，通过指令passwd设置密码。
  9. 毕竟爱折腾，难免会把子系统环境(lxss目录)玩坏掉，因此干正事前最好先备份下以便快速还原。注意，不要直接右键复制或者打包，可能会导致文件权限丢失的。xcopy %localappdata%\lxss %localappdata%\lxss.bak /E
  10. 当然，如果你比较任性也可以不执行上一步的备份操作，通过命令行运行lxrun /uninstall /full轻松卸载子系统，重复上面的步骤即可重装，不过要注意下载速度时好时坏哦。

查看Ubuntu的版本

  1. 使用命令：cat /proc/version 查看(linux内核版本)
  2. 使用命令：uname -a 查看
  3. 使用命令：lsb_release -a 查看(ubuntu版本)

设置更换数据源

在Ubuntu下我们可以通过apt-get命令很方便的安装/卸载软件，由于默认的软件包仓库是位于国外的，安装软件的时候就可能遇到各种网络问题或者下载到的一些资源不完整，因此就需要切换数据源为国内的镜像站点来改善。

X11图形界面

Linux作为服务器使用，基本好了，但是有时候我们还是想看看Linux下的图形界面的。那么就要依靠X11了。
首先安装Xming;

然后再windows宿主机器上安装Xming-6-9-0-31-setup，安装好以后，运行Xming，回到Shell，执行

  1. 安装完直接打开Xming即可
  2. 安装一个firefox测试
    apt-get install firefox
  3. 运行(在程序指令前加上”DISPLAY=:0 “)
    DISPLAY=:0 firefox
  4. 简化配置
    每次运行程序都要输入DISPLAY=:0挺累的，执行下列指令后重启bash即可省这个步骤
    echo “export DISPLAY=:0.0” >> ~/.bashrc
