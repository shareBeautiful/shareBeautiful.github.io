vs code编辑器

vs code 插件

  1. prettier 是目前 Web 开发中最受欢迎的代码格式化程序
  2. ESlint 接管原生 js 提示，可以自定制提示规则。
  3. beautify 格式化代码的工具
  4. Minify 这是一款用于压缩合并 JavaScript 和 CSS 文件的应用程序。
  5. Path Intellisense 自动路劲补全，默认不带这个功能的
  6. Debugger for Chrome 让 vscode 映射 chrome 的 debug功能，静态页面都可以用 vscode 来打断点调试，
  7. jQuery Code Snippets jquery 重度患者必须品
  8. JavaScript(es6)code snippets ES6语法提示
  9. Atuo Rename Tag 修改 html 标签，自动帮你完成尾部闭合标签的同步修改
  10. Open-In-Browser 在浏览器中打开文件
  11. Better Comments 编写更加人性化的注释（推荐）
  12. View In Browser 快读打开浏览器Ctrl+F1
  13. Sass sass插件
  14. filesize 在底部状态栏显示当前文件大小，点击后还可以看到详细创建、修改时间
  15. JavaScript (ES6) code snippets 常用的类声明、ES 模块声明、CMD 模块导入等
  16. IntelliSense for CSS class names CSS 类名补全，会自动扫描整个项目里面的 CSS 类名并在你输入类名时做智能提示
  17. Setting Sync 可同步包含的所有扩展和完整的用户文件夹

插件推荐

  1. Key promoter x

    Key promoter这款插件适合新手使用。当你点击鼠标一个功能的时候，可以提示你这个功能快捷键是什么。
    这是一个非常有用的功能，很快就可以熟悉软件的快捷功能了。 如果有快捷键的，会直接显示快捷键

  2. AceJump

    Ctrl+; 快捷键，激活插件。此时，IDE代码编辑区会出现一个蓝色光标，然后输入你要跳转的标签shift+enter跳转

  3. CodeGlance

    代码预览滚动条

  4. Coderpillr Icons

    项目图标
    Atom Material Icons 项目图标

  5. A8Translate

    翻译插件，选择单词alt+t启动，

  6. IdeaVim

    基于IntelliJ的Vim仿真插件

  7. LiveEdit

    提供实时编辑 HTML / CSS / JavaScript的工具。

  8. activate-power-mode

    装逼插件，不解释不解释

常用快捷键

  1. Ctrl + /: 单行注释
  2. Ctrl + Shift + /: 添加注释文本/块注释
  3. Ctrl + F: 查找文本
  4. Ctrl + R: 替换文本
  5. Ctrl + D: 复制行
  6. Ctrl + 鼠标单击： 跳到变量声明处
  7. Ctrl + G: 查找行
  8. Ctrl + N: 查找类
  9. Ctrl + Shift + N: 查找文件
  10. Ctrl + Shift + I: 显示当前CSS选择器或者JS函数的详细信息
  11. Ctrl + Shift + up/down: 代码向上或向下移动
  12. Ctrl + Alt + L: 格式化代码
  13. Alt + 7: 显示当前文件的结构
  14. Shift + F6: 重构-重命名
  15. Alt + F12: 调出terminal
