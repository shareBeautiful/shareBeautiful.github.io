gulp使用笔记

发表于 2017-09-04 | 更新于 2021-08-11 | 分类于 node
本文字数： 27k | 阅读时长 ≈ 49 分钟

介绍：

Gulp是Web开发中帮你自动完成任务的工具。它可以完成的前端任务包括：
启动Web Server
当文件修改保存后，自动刷新浏览器
批量处理Sass或者Less文件
优化资源CSS, JavaScript, 和 images
Gulp能做的比以上更多。你甚至可以用Gulp创建一个静态站点生成器。Gulp真是非常强大的。下面我们来学习创建自己的Gulp任务。

目前最流行的构建工具有Gulp和Grunt.
Gulp和Grunt的主要区别在于配置工作流的不同。Gulp的配置更加简洁，运行速度更快。

安装顺序：

  1. 安装node
  2. 全局安装gulp
  3. 初始化项目 npm init
  4. 本地安装gulp
  5. gulp插件安装

安装gulp

1.全局安装gulp

$ npm install gulp -g

2.初始化

$ npm init

3.本地安装gulp

$ npm isnntall gulp --save-dev

4.在目录中创建一个gulpfile.js文件

gulpfile文件是配置gulp的插件使用文件

gulp插件安装

安装 gulp-sass

$ npm install gulp-sass --save-dev

安装 gulp-imagemin 图片压缩

$ npm install gulp-imagemin --save-dev

安装 gulp-babel es6转化es5

npm install babel-cli babel-preset-env --save-dev

安装 browser-sync 实现热加载

$ npm install browser-sync --save-dev
