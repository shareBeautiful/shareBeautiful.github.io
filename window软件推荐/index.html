window 软件推荐

发表于 2017-09-27 | 更新于 2021-08-11 | 分类于 Windows
本文字数： 27k | 阅读时长 ≈ 49 分钟

个人在使用windows过程中的一些实用性软件推荐。

windows 扩展工具

1. Listary

Listary是一款用于Windows的文件名定位/搜索辅助软件。它为Windows传统低效的文件打开/保存对话框提供了便捷、人性化的文件（夹）定位方式，同时改善了常见文件管理器中文件夹切换的效率。

下载地址：Download

2. Fences3

Fences是一个Windows平台下的桌面管理软件，简洁实用，可以有效的管理桌面应用程序和文件。

下载地址：Download

3. Wox

在日常的电脑操作中，我们常常花很多时间重复做着一件事情：退回桌面/开始菜单->寻找软件图标->点击启动。但如果你喜欢高效的[键盘]操作，那么你需要一款好用的快速启动器软件。Listary比起Wox功能上更专注于文件的搜索

下载地址：Download

4. Everything

Everything是voidtools开发的一款文件搜索工具

下载地址：Download

5. Clover

Clover 就是给资源管理器增加类似谷歌 Chrome 浏览器的多标签页功能，软件本身是免费的，当然也少不了双击关闭标签页功能

下载地址：Download

6. Total Command

Total Commander 是一款应用于 Windows 平台的文件管理器 ，它包含两个并排的窗口，这种设计可以让用户方便地对不同位置的“文件或文件夹”进行操作，例如复制、移动、删除、比较等，相对 Windows 资源管理器而言方便很多，极大地提高了文件操作的效率，被广大软件爱好者亲切地简称为：TC
它拥有文件快速预览、快速搜索、多标签、文件比较、批量重命名、FTP 客户端等诸多实用的功能，并可通过大量的插件进行个性化配置。

下载地址：Download

7. Directory Opus

一个功能非常强大的文件管理工具，程序采用和windows资源管理器相似的界面。
同TC类似，不过多用与鼠标操作

下载地址：Download

8. AutoHotkey

AutoHotkey是一个小工具软件，可以简化你的重复性工作,只需提前写好一个脚本，一键自动化启动或运行程序，甚至帮你自动登陆某个论坛，这里也推荐善用佳软的教程：AutoHotkey 0级入门教程:让重复工作一键完成

下载地址：Download

9. cmder

至少要比Windows原生Cmd好出了天际。因为好用，所以“必备”
她把ConEmu ，msysgit和clink打包在一起，让你无需配置就能使用一个真正干净的Linux终端！她甚至还附带了漂亮的monokai配色主题。）;作为一个压缩档的存在, 可即压即用。你甚至可以放到USB就可以虽时带着走，连调整过的设定都会放在这个目录下，不会用到系统机码(Registry)，所以也很适合放在Dropbox / Google Drive / OneDrive共享于多台电脑。
安装和设置

下载地址：Download
类似：
ConEmu
PowerCmd
ConsoleZ
Babun

10. Glary Utilities

Glary Utilities 一站式电脑优化解决方案。它允许你清理系统垃圾文件，无效的注册表键值，上网记录，
管理和删除IE插件，分析磁盘空间使用情况，查找重复文件，优化内存，查找、修理或删除快捷方式，
管理windows启动程序，卸载软件，安全删除文件，查找空目录,右键菜单管理，包括超过20
+系统实用程序来提高你的计算机的性能。支持简体中文等多语系界面。

下载地址：Download

11. QuickLook/Seer

它可以为 Win 增加与 Mac 相似的空白键快速预览功能

下载地址：QuickLook Seer

12. 火萤酱

类似于listary

下载地址：火萤酱

windows10 相关优化设置

解决Task Scheduler造成CPU占用100%问题

在此电脑右键选择管理-计算机管理-系统工具-任务计划程序（也可以从控制面版里的管理工具进入，不过我喜欢直接从此电脑右键选择管理进入，方便。）
定位到：任务计划程序库\Microsoft\Windows\customer experience improvement program
可以看到这里有三个计划任务，都是微软的客户体验改差计划!全部选择禁止或者删除，这样下次电脑再启动之后就不会有这三个计划了。

关闭Windows Defender安全中心

  1. 在Cortana搜索栏中输入regedit，按回车键进入注册表编辑器。
  2. 定位到HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\SecurityHealthService
  3. 在右侧找到DWORD（32位）值，命名为Start。
  4. 修改数值数据为4。
  5. 重启文件资源管理器，或注销再登录/重启系统。

解决Win10 CPU占用高：

http://www.pconline.com.cn/win10/928/9287464.html
Defender本身不应该会占据如此之多的CPU资源，可能和你目前所安装的某些软件有冲突，从而导致了目前的问题。
你能大致介绍下你安装过哪些软件么？特别是安全类软件。
同时，你可以尝试下以下步骤，暂时禁用一些第三方程序，确认结果：

  1. 点击开始，在搜索程序和文件夹中输入msconfig，
  2. 点击服务，选择隐藏所有Microsof服务，然后点击全部禁用。
  3. 点击启动，点击全部禁用。
  4. 点击应用，点击确定，重新启动后再次检查问题。
